# 기능적 요구사항 - MVP

## 회원 관리
1. 사용자는 회원가입할 수 있다
2. 사용자는 로그인할 수 있다
3. 시스템은 회원가입 시 개인 문서함을 자동으로 생성한다
4. 시스템은 개인 문서함 생성시 총 사용량을 할당한다
5. 시스템은 인증된 사용자 기준으로 파일/폴더 접근을 제어한다

## 문서함 관리
1. 사용자는 하나의 개인 문서함을 가진다
2. 개인 문서함은 사용자가 업로드한 파일 및 폴더를 관리한다
3. 사용자는 개인 문서함에서 자신의 파일 및 폴더 목록을 조회할 수 있다
4. 문서함은 사용량을 가진다
5. 시스템은 문서함 최초 생성시 사용량을 0으로 초기화 한다
6. 시스템은 문서함 최초 생성시 총 사용량을 50GB로 초기화한다 

## 파일/폴더 관리
1. 사용자는 파일을 생성할 수 있다
2. 사용자는 폴더를 생성할 수 있다
3. 사용자는 특정 폴더의 하위 목록을 조회할 수 있다
4. 사용자는 파일 및 폴더를 이동할 수 있다

## 파일 업로드
1. 사용자는 업로드 URL을 발급받을 수 있다
2. 사용자는 파일을 업로드할 수 있다
3. 시스템은 파일 업로드 상태를 관리한다
4. 시스템은 업로드 완료 시 상태를 갱신한다
5. 시스템은 파일을 업로드한 사용자를 소유자로 지정한다

## 파일 다운로드
1. 사용자는 파일을 다운로드할 수 있다

## 파일/폴더 삭제
1. 사용자는 파일 및 폴더를 삭제할 수 있다
2. 시스템은 삭제된 파일 및 폴더를 휴지통으로 이동시킨다

## 휴지통 관리
1. 사용자는 휴지통에 있는 파일 및 폴더를 조회할 수 있다
2. 사용자는 휴지통에서 파일 및 폴더를 복구할 수 있다
3. 사용자는 휴지통에서 파일 및 폴더를 영구 삭제할 수 있다

## 권한 관리
1. 사용자는 파일 및 폴더에 대한 접근 권한을 가진다
2. 사용자는 권한이 있는 리소스만 접근할 수 있다
3. 시스템은 리소스 접근 시 권한을 검증한다
4. 소유자는 해당 리소스에 대한 관리 권한을 가진다****
